Deploy aplicação Node js com Git , Heroku



O passo a passo



Pre requisitos

Já eve ter criado sua conta no GitHub, Heroku

a) Criei a pasta do projeto
b) depois criei uma pasta com o mesmo nome no heroku create projetopassos
c) Iniciei um projeto simples com express npm init –yes
d) Criei uma extrututa MVC
scr
public
routes
viwes
partials

e) utilizei as bibliotecas npm i express ejs morgan
npm i nodemon -D
f) Criei o ..gitignore para não subir a pasta node_modulos
g) Outro detalhe esta no package.json


              h) Criei Procfile , que é um detalhe importante este arquivo dentro dele coloquei
web: npm start


Logo criei algumas viwes, rotas e montei um servidor de blog, poderia ser qualquer tipo de modelos projeto. Pedido, Frente de loja etc, criei o blog para disponibilizar meus testes

i) Foi no github criei a pasta iniciei o git init na maquina
j) fiz o link do projeto local com o github
git remote add origin https://github.com/UrsulaASobrinho/projetopassos.git
git add .
git commit -m "adciona projeto no horoku"
git push -u origin master


k) vá no heroku opção deploy , escolhar git e coloquei o mesmo link que utilizou para remote e click na opção atualizar automaticamente
Mais a baixo na mesma tela click em fazer o primeiro deploy manualmente Pegue o link fornecido pelo heroku e abrano brouwse
https://projetopassos.herokuapp.com
Feito!
